# 🔢 Contador

Aplicação simples de contador feita em **JavaScript**.  
Permite incrementar, decrementar e zerar valores.

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript

## 🎯 Funcionalidades
- Incrementar valor
- Decrementar valor
- Resetar contador

## ▶️ Como executar
Abra o arquivo `index.html` no navegador.

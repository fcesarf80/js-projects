# 🧮 Calculadora

Calculadora básica feita em **JavaScript**.  
Permite realizar operações matemáticas simples diretamente no navegador.

## 🚀 Tecnologias
- HTML
- CSS
- JavaScript

## 🎯 Funcionalidades
- Adição
- Subtração
- Multiplicação
- Divisão
- Limpar resultado

## ▶️ Como executar
Abra o arquivo `index.html` no navegador.
